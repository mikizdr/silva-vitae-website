## GitHub Pages (static HTML site)

This repository also includes a multi-page static HTML website that can be published with GitHub Pages.

### Site structure

- `index.html`
- `about.html`
- `contact.html`
- `lepljenje-pločica.html`
- `docs/index.html`
- `docs/getting-started.html`
- `docs/page2.html`
- `assets/css/styles.css`
- `assets/js/main.js`

### Publish steps

1. Push changes to `main`.
2. Open **GitHub → Settings → Pages**.
3. Under **Build and deployment** choose:
	- **Source**: `Deploy from a branch`
	- **Branch**: `main`
	- **Folder**: `/ (root)`
4. Save and wait for deployment.
